When to Use Each Style

Best Practices

  • Bar: Blog posts, documentation, long-form content
  • Circle: Landing pages, minimal designs, corner placement
  • Rail: Editorial sites, vertical emphasis, full-height
  • Counter: Technical docs, precise feedback, data-driven

Accessibility Notes

  • All indicators use aria-hidden="true" (decorative)
  • CSS provides static 100% state for reduced motion
  • No keyboard interaction required (passive visualization)
  • Works with any smooth scroll library or native scroll
